## вңҰ What is NeuroGest?

NeuroGest is a browser-based gesture playground that tracks your hands through your webcam and turns them into meaningful input вҖ” no hardware, no plugins, no setup beyond pointing a camera at yourself.

Two experiences live inside it:

- **вңҚпёҸ Air Drawing** вҖ” paint strokes in the air with your index finger, drag them around with a pinch, undo with a peace sign
- **рҹӘҗ Solar System** вҖ” navigate a live 3D orrery, rotate planets with your hand, zoom in, and tap to inspect every world

Everything runs locally in your browser. Nothing is recorded or sent anywhere.

## вңҰ Features

| | |
|---|---|
| рҹ–җпёҸ **Two-hand tracking** | MediaPipe tracks both hands at 30 fps вҖ” drawing hand and control hand work independently |
| вңҚпёҸ **Air Drawing** | Paint strokes with your index finger; solid, dashed, or dotted; 8 brush colors; glitter mode |
| рҹӨҸ **Pinch-to-drag** | Pinch near any stroke to grab and reposition it |
| вҶ©пёҸ **Gesture undo & clear** | Peace sign to undo; open palm held 1.5 s to wipe the canvas |
| рҹӘҗ **3D Solar System** | Real orbital speeds and sizes; Saturn's rings; smooth camera momentum |
| рҹ”ӯ **Planet detail modals** | Click any planet to open a mini 3D viewer with real facts |
| рҹӨҹ **Panel toggle** | Three-finger gesture opens/closes the side panel from either hand |
| рҹ“· **Camera overlay modes** | Show your feed fully, dimmed, or hidden while still tracking |
| рҹ’ҫ **Stroke persistence** | Your drawing survives page refreshes (cleared on tab close) |
| вҶ“ **PNG export** | Save your artwork as a full-resolution PNG in one click |

## вңҰ Gesture Reference

### вңҚпёҸ Air Drawing

| Gesture | Action |
|---|---|
| вҳқпёҸ Index finger extended | **Pen down** вҖ” draw a stroke |
| вңҠ Fist / lower index | **Pen up** вҖ” stop drawing |
| рҹӨҸ Pinch near a stroke | **Grab & drag** that stroke |
| вңҢпёҸ Peace sign (hold 0.5 s) | **Undo** last stroke вҖ” either hand |
| рҹ–җпёҸ Open palm (hold 1.5 s) | **Clear** all strokes вҖ” either hand |
| рҹӨҹ 3 fingers (mid + ring up) | **Toggle** side panel вҖ” either hand |

### рҹӘҗ Solar System

| Gesture | Action |
|---|---|
| рҹӨҸ Pinch + move | **Rotate** the scene |
| рҹӨҸ Pinch + spread/close | **Zoom** in and out |
| рҹӨҸ Tap a planet (quick pinch-release) | **Open** planet detail |
| рҹ–җпёҸ Open palm | **Reset** camera to default view |
| рҹӨҸ+рҹӨҸ Both hands pinch + spread | **Two-hand zoom** |

> **Mirror note:** the camera feed is flipped like a mirror. Your left hand appears on the right side of the screen вҖ” just move naturally.

## вңҰ How It Works

```
Webcam
  в””в”Җв–ә MediaPipe Tasks Vision (WASM, off main thread, 30 fps)
        в””в”Җв–ә useHandTracking  вҶ’  { left: Landmark[21], right: Landmark[21] }
                в””в”Җв–ә useGesture(hand)          per-hand: pinch, peace, index, fistвҖҰ
                в””в”Җв–ә useTwoHandGesture(l, r)   cross-hand: two-hand pinch/zoom
                        в”ңв”Җв–ә /draw  вҶ’  Canvas 2D вҖ” strokes, particles, cursors
                        в””в”Җв–ә /solar вҶ’  Three.js вҖ” rotation, zoom, raycaster
```

Gesture classification uses **exponential moving average (EMA) smoothing** with **hysteresis** (separate ON / OFF thresholds) to prevent rapid flicker between states. Hold-based gestures (peace sign, open palm) use a timer that resets if the gesture breaks, so accidental frames don't trigger them.

## вңҰ Deployment

The app is configured for **Vercel** with a SPA rewrite rule so `/draw` and `/solar` deep-links work correctly.

```bash
# Deploy via Vercel CLI
npx vercel

# Or connect your GitHub repo at vercel.com for automatic deploys
```

`vercel.json` already contains:
```json
{
  "rewrites": [{ "source": "/(.*)", "destination": "/index.html" }]
}
```
